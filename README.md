# LINE Chatbot

## Local Development

1. Create a virtual environment

```bash
python -m venv venv
```

2. Activate the virtual environment

   for Linux and MacOS:

   ```bash
   source venv/bin/activate
   ```

   for Windows:

   ```cmd
   venv\Scripts\activate.bat
   ```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python main.py
```
